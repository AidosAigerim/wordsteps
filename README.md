# 📚 WordSteps — английский по шагам

Сайт для изучения английского языка без грамматики: 1000 самых нужных слов, чтение с транскрипцией, слушание и игры.

## Что внутри

- **3 уровня**: A1 (300 слов), A2 (300 слов), B1 (400 слов) — по 15 уроков в каждом
- **4 игры в каждом уроке**: выбор перевода, обратный перевод, слушание, сборка слова из букв
- **15 историй для чтения** — написаны только из слов уроков; любое слово в тексте кликабельно (перевод + произношение), истории можно слушать
- **Мотивация**: XP, серия дней 🔥, 17 достижений
- **Профили учеников** со статистикой: точность, трудные слова, история уроков
- **Режим преподавателя** (PIN): просмотр всех уроков, открытие уроков ученикам, статистика каждого ученика, импорт кода прогресса

## Запуск

Сайт полностью статический — просто откройте `index.html` в браузере,
или запустите локальный сервер: `node serve.js` → http://localhost:4173

Прогресс хранится в браузере (localStorage). Озвучка — Web Speech API (встроена в браузер).
